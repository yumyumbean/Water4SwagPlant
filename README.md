
# THE LAZY BUM PLANT WATERER
A plant watering device to help you water your plant automatically/wirelessly. (you lazy bum)
## GETTING STARTED
JUST PLUG 'ER IN. Everything is automatic, the code will need to be adjusted with your very own credentials and Adafruit dashboard so you can monitor your stats.
## STATS
The Adafruit dashboard will keep track of Temperature, Air Pressure, Humidity, Soil Moisture, and Air Quality. The dashboard you will need to create will also feature a button that will water your plant for the perfect amount of time
## DASHBOARD
Create feeds and a dashboard at Adafruit.io
